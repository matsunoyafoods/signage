🚀 遠隔更新の手順

  1. 下で時間帯モード・画像・秒数を編集する
  2. 「config.json を書き出す」でファイルを保存
  3. GitHub（matsunoyafoods/signage）で config.json を差し替え（Upload）
  4. 約30秒で全サイネージが自動反映（URLは固定）
秒ごとにサイネージが設定を確認

⚠ アップロード画像は config.json に埋め込まれます。合計サイズ: 0（重い場合は画像URLの利用を推奨）

🎵 BGM（全時間帯 共通）

70%

音楽は config.json に埋め込めないため、mp3等のファイルは GitHub リポジトリの music/ フォルダに置き、 music/ファイル名.mp3 のように指定します。曲は上から順にループ再生されます。 著作権に配慮し、店舗利用可（ロイヤリティフリー等）の音源をご利用ください。再生はサイネージで「開始」をタップした後に始まります。

📅 特別休日（土日祝あつかいにする日）

日本の祝日は自動で「土日祝」判定されます。ここには、祝日ではないけれど休業・特別営業にしたい日（年末年始・お盆など）を追加します。

時間帯が重なる場合は「上にあるモードが優先」されます。時間外はどのモードにも該当せず待機画面になります。 編集内容はこのPCに自動保存されます（保存＝config.json書き出し→GitHub差し替えで反映）。
